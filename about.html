The movie DB

api documentation

The movie db api provide films and television series data sets. We use it to display up to date information on poster image, release date, film description, rating, and actors, which we get them in JSON format. Moreover, the API supply XML format too and require an API key to access data. The api data set contains a huge amount of specific data on tv content.


The movie db is used in app pages:

  • index.html
  • detail.html

OpenWheather

api documentation

OpenWeather is an online weather data and forecast provider, it utilize meteo broadcast services raw data from airport weather stations, raw data from radar stations, and raw data from other official weather stations. OpenWeather provide JSON, XML and HTML endpoints. We use these data to help our customers chose the best cinema experience they can get, like staying home, going to theater, or drive-in theater.
